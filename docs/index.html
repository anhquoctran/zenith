Powered by Native Technologies

Designed for absolute performance and seamless user experience across operating systems.

FFmpeg Native Engine

Direct low-level C++ interop with FFmpeg 6.x and DirectShow/V4L2 for hardware-accelerated video encoding with zero overhead.

Avalonia UI

Write once, run everywhere. A sleek and responsive interface running on .NET 10.

Region Selection

Precisely capture the area you need with an intuitive drag-to-select overlay.

Picture-in-Picture Webcam

Overlay your hardware webcam directly into the corner of your screen recording. Fully resizable and dynamically scaled.

Floating Widgets

When you start recording, Zenith gracefully steps out of your way and provides a non-intrusive floating control widget that stays on top.
